songs.dapper.nz

I created this website for my Church's worship team. It's designed to create and edit worship songs while also storing the song in the URL. This makes for an extermely long url (2000 characters in some cases) but makes the song you create somewhat easily shared.

It is a work in progress and you cannot create songs just yet, only edit them. This website has be very helpful in developing my JS skills as I had not used it much before then.

Papa's book

This website came about when my Grandad (Papa) passed. My Dad found a folder on his computer dating back to 2007 where Papa had started writing an autobiography titled "60 Years of School Holidays".

I'm currently in the process of putting the 15 chapters of this book on the internet so anyone curious can read it. I'm sad I didn't find out about it when he was still alive, as he had some files with notes on making a website which I would've loved doing with him.
